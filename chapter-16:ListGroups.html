List Group With Linked Items

Instead of plain text items (<li>), you use anchor tags (<a>) or buttons inside .list-group to make each item clickable. To create a list group with linked items, use <div> instead of <ul> and <a> instead of <li>. Optionally, add the .list-group-item-action class if you want a grey background color on hover:

First item Second item Third item

The .disabled class adds a lighter text color to the disabled item. And when used on links, it will remove the hover effect:
