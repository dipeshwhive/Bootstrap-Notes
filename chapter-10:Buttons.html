Block Level Buttons

A block level button spans the full width of its parent element.

In Bootstrap 5, you can create a full-width button by placing it inside a container with the .d-grid class.

The .d-gird class creates a grid container that allows the button to stretch across the entire width.

A button can be set to an active (appear pressed) state using the class .active The disabled attribute makes a button unclickable. Note that <a> elements do not support the disabled attribute and must therefore use the .disabled class to make it visually appear disabled.
