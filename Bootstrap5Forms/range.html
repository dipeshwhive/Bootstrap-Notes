Bootstrap 5 Form Range - Complete Notes

1. Custom Range Slider

Range sliders allow users to select a value from a range by dragging a slider. To style a range input in Bootstrap 5, add the .form-range class to the <input> element with type="range".

Basic Syntax:

Apply .form-range class to a range input element to get Bootstrap's styled range slider.

  • Use .form-range class on the range input element

  • By default, range sliders are full-width within their container
  • The default value range is 0 to 100
  • The default starting value is 50 (the middle)
  • The default step/interval is 1

2. Range Steps

By default, the interval (step) between the range values is 1. You can change this using the step attribute to increase or decrease the granularity of the slider increments.

Understanding Steps:

  • step="1" - Default: increments by 1 for each position on the slider
  • step="5" - Increments by 5 (0, 5, 10, 15, 20...)
  • step="10" - Increments by 10 (0, 10, 20, 30...)
  • step="0.5" - Allows decimal values with 0.5 increments
  • step="0.1" - Allows precise decimal increments

Key Points:

  • The step attribute controls the granularity of the slider
  • Larger step values make the slider jump in bigger increments
  • Smaller step values allow for more precise selections
  • Step can be decimal values like 0.1 or 0.5
  • Choose appropriate step values based on your use case (e.g., prices, volumes, percentages)

3. Min and Max Values

By default, the minimum value is 0 and the maximum value is 100. You can change these using the min and max attributes to set custom ranges.

Understanding Min and Max:

  • min="0" - Default minimum value (lowest possible selection)
  • max="100" - Default maximum value (highest possible selection)
  • You can set any numeric range you need
  • min and max can be used together or separately
  • Default starting position is the midpoint between min and max

Key Points:

  • Use min attribute to set the lowest selectable value
  • Use max attribute to set the highest selectable value
  • The default min is 0 and default max is 100
  • You can use negative values for min (e.g., temperature ranges)
  • The slider will automatically adjust its visual representation based on the range
  • Combine with step for appropriate increments

Best Practices for Bootstrap Range Sliders

  • Always use .form-range class on range input elements
  • Always associate a .form-label with a range input using id and for attributes
  • Choose appropriate min and max values for your use case
  • Select an appropriate step value - not too small (overwhelming choices) or too large (missing important values)
  • Display the current selected value to give users immediate feedback (using JavaScript)
  • Use meaningful labels that explain what the range controls (e.g., "Price", "Quality", "Volume")
  • Add helpful text below the slider explaining the range and units
  • Consider providing preset buttons in addition to the slider for common values
  • Test on mobile devices - ensure the slider is easy to use with touch
  • Use appropriate units in the label (e.g., $, %, °C, hours, etc.)
  • For price ranges, update the display value in real-time as the user drags the slider
  • Wrap range sliders in form groups with proper spacing using margin classes

JavaScript to Update Values:

The input event listener fires every time the VALUE of an input element changes.

// Update price value
document.getElementById('filterPrice').addEventListener('input', function() {
  document.getElementById('displayPrice').textContent = this.value;
  document.getElementById('displayPriceSmall').textContent = this.value;
});

// Update rating value
document.getElementById('filterRating').addEventListener('input', function() {
  document.getElementById('displayRating').textContent = this.value;
  document.getElementById('displayRatingSmall').textContent = this.value;
});

// Update discount value
document.getElementById('filterDiscount').addEventListener('input', function() {
  document.getElementById('displayDiscount').textContent = this.value;
  document.getElementById('displayDiscountSmall').textContent = this.value;
});
