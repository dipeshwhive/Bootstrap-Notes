Bootstrap 5 Forms - Complete Notes

1. Stacked Form (Vertical Forms)

By default, Bootstrap displays form elements vertically (stacked). Key styling rules:

  • .form-control - Applied to input and textarea elements for proper styling
  • .form-label - Applied to label elements for correct padding and alignment
  • .mb-3 - Adds bottom margin to form groups for spacing
  • .form-check - Container for checkboxes and radio buttons
  • .form-check-input - Class for checkbox/radio input elements
  • .form-check-label - Class for checkbox/radio labels

4. Form Control Size

Control the size of form inputs using size modifier classes:

  • .form-control-lg - Creates a large input field
  • .form-control - Default normal-sized input field
  • .form-control-sm - Creates a small input field

5. Disabled and Readonly Inputs

Use HTML attributes to prevent user interaction with form fields:

  • disabled - Disables the input field completely (grayed out appearance)
  • readonly - Allows reading but prevents editing the input field

6. Plain Text Inputs

Use the .form-control-plaintext class to style input fields without borders while maintaining proper margins and padding. This is useful for displaying data that looks like form content but isn't editable.

Key Bootstrap Form Classes Summary

Class Purpose
.form-control Styles input and textarea elements
.form-label Styles label elements with proper padding
.form-check Container for checkboxes and radio buttons
.form-check-input Styles checkbox and radio button inputs
.form-check-label Styles labels for checkboxes and radio buttons
.form-control-lg Creates large form control
.form-control-sm Creates small form control
.form-control-plaintext Removes borders while maintaining padding
.form-control-color Styles color picker inputs
.mb-3 Adds bottom margin for spacing

Best Practices for Bootstrap Forms

  • Always wrap form elements in appropriate containers with spacing classes like .mb-3
  • Use .form-label class on all label elements for consistent styling
  • Apply .form-control to text inputs and textarea elements
  • Use the grid system (.row and .col) for inline forms
  • Ensure proper contrast and readability with color selections
  • Use size modifiers consistently throughout your form
  • Test form accessibility with keyboard navigation and screen readers
