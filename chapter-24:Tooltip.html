Bootstrap 5 Tooltip

The Tooltip component is a small pop-up box that appears when the user moves the mouse pointer over an element.

How to create a tooltip

To create a tooltip, add the data-bs-toggle="tooltip" attribute to an element.

Use the title attribute to specify the text that should be displayed inside the tooltip:

Positioning Tooltips

By default, the tooltip will appear on top fo the element.
