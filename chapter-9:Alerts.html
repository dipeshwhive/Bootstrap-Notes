Bootstrap 5 Alerts

Alerts

Bootstrap 5 provides an easy way to create predefined alert messages:

Success! This alert box could indicate a successful or positive action.
Info! This alert box could indicate a neutral informative change or action.
Warning! This alert box could indicate a warning that might need attention.
Danger! This alert box could indicate a dangerous or potentially negative action.
Primary! Indicates an important action.
Secondary! Indicates a slightly less important action.
Dark! Dark grey alert.
Light! Light grey alert.

Alerts are created with the .alert class, followed by one of the contextual class .alert-success, .alert-info, .alert-warning, .alert-danger, .alert-primary, .alert-secondary, .alert-light or .alert-dark.

Alert Links

Add the .alert-link class to any links inside the alert box to create "matching colored links":

Success! You should read this message.

Closing Alerts

To close the alert message, add a .alert-dismissible class to alert container. Then add class="btn-close" and data-bs-dismiss="alert" to a link or a button element (when you click on this the alert box will disappear).

Success! This alert box could indicate a successful or positive action.
