Float

Use the .float-start class to float an element to the left, and .float-end to float it to the right. The floated element will wrap text around it.

Floated left

Lorem ipsum dolor sit amet, consectetur adipiscing elit. The text wraps around the floated element.

Clearfix

The .clearfix class clears floated elements. When you have floated elements, the parent container may collapse because the floated elements are removed from the normal document flow. Use clearfix on the parent to prevent this collapse and make the parent expand to contain the floated children.

Floated element

Without clearfix, the parent would collapse. With clearfix, the parent properly contains the floated element.

Responsive Floats

Use the .float-sm-start|end, .float-md-start|end, .float-lg-start|end, .float-xl-start|end, .float-xxl-start|end classes to control float on specific screen sizes. This allows elements to float on larger screens but appear normally on smaller screens.

Floats left on md and larger

Spacing (Margin & Padding)

Bootstrap provides utility classes for margins (m*) and padding (p*) that range from 0 to 5 (or more). The classes follow the format: .m-{size}, .p-{size}, .mt-{size}, .mb-{size}, .ml-{size}, .mr-{size}, .pt-{size}, .pb-{size}, .pl-{size}, .pr-{size}.

Margin Examples

m-5 (3rem)

Padding Examples

Directional Spacing (Top, Bottom, Left, Right)

Use .mt-{size} for top margin, .mb-{size} for bottom margin, .ms-{size} for left margin, .me-{size} for right margin, and similar for padding.

mt-3 (margin-top)
mb-3 (margin-bottom)
ms-3 (margin-left)
me-3 (margin-right)

Responsive Spacing

Use the .m{property}-{breakpoint}-{size} format to apply spacing at specific screen sizes. For example, .mt-sm-2, .p-md-3, .mb-lg-5.

Padding changes based on screen size (responsive)

Aspect Ratio

Use the .ratio class combined with .ratio-16x9, .ratio-4x3, .ratio-1x1, .ratio-21x9 to create responsive aspect ratio containers. This is useful for maintaining proportions of images or videos.

Visually Hidden Class

The .visually-hidden class hides an element visually while keeping it accessible to screen readers. This is useful for providing additional context to users with assistive technologies without cluttering the visual interface.

The text "about our services" is not visible but will be read by screen readers for context.
