The .d-grid class creates a CSS Grid container. By default, it creates a single column layout. All direct children become grid items.

Auto-Fit Grid

Use repeat(auto-fit, minmax(...)) to create a flexible grid that automatically adjusts the number of columns based on available space. This is useful for responsive layouts that don't require specific breakpoints.

Grid vs Flexbox

Both CSS Grid and Flexbox are layout tools, but they serve different purposes:

  • Grid: Two-dimensional layout system. Great for overall page layouts with rows and columns. Aligns items along both axes simultaneously.
  • Flexbox: One-dimensional layout system. Great for components and smaller layouts. Aligns items along a single axis (row or column).
  • When to use Grid: Page layouts, dashboard layouts, card grids, complex multi-row/column arrangements.
  • When to use Flexbox: Navigation bars, button groups, single-line layouts, component alignment within a page.
