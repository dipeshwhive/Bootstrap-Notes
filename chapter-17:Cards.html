Bootstrap 5 Cards

Cards in a Bootstrap 5 is a bordered box with some padding around its content. It includes options for headers, footers, content, colors, etc.

Titles, text, and links

Use .card-title to add card titles to any heading element. The .card-text class is used to remove bottom margins to a <p> element if it is the last child (or the only one) inside .card-body. The .card-link class adds a blue color to any link, and a hover effect.

Card Images
